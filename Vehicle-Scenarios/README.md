## Vehicle Scenario Custom Extension Import

1. **Clone the repository**
    
    `git clone https://[YOUR_REPO_URL]`
    
2. Add the **parent directory** of the cloned repository to the **Extension Search Paths** inside the Extensions window.
    - *(Navigate to: Extensions > Gear Icon (Settings) > Extension Search Paths)*
3. Verify that **"SIX VEHICLE SCENARIOS"** appears under **Extensions > THIRD PARTY > User**.
